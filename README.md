# Visual Navigation and Safety Assistance System for Logistics Orientrack

This project is my BSc Thesis for BME, Department of Automation and Applied Informatics. 

## Motivation
Standard navigation often fails in regions like Central Asia (Kazakhstan and Kyrgyzstan) due to the lack of precise addresses in markets and private districts. Drivers usually find the general area but struggle to find the exact gate or container. Also, mountainous terrain leads to signal loss, which is a safety risk.

## Objectives
The system uses a driver's smartphone as the main tracking device. It includes a Telegram bot interface for the driver and a web dashboard for the dispatcher.

Main goals:
* Real-time vehicle tracking and status updates.
* Visual Anchors: Using photos and GPS to mark exact unloading points.
* Safety monitoring to detect when a driver loses connection.
* Visualization using OpenStreetMap and Leaflet.js.
